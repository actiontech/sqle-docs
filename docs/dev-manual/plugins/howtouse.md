# 数据库审核插件使用说明

## 概述
SQLE 提供了丰富的数据库审核插件，用于支持不同类型数据库的 SQL 审核功能。这些插件已经默认集成在 SQLE 中，无需额外配置即可使用。

## 支持的数据库类型
请参考 [支持的数据源类型](/docs/support/compare.md)

## 插件使用方式

### 1. 默认插件
当前数据源插件已默认集成在 SQLE 中，无需配置，开箱即用。在添加数据源时，只需选择对应的数据库类型即可。

### 2. 自定义插件
如果需要使用自定义的审核插件，请参考[数据库审核插件开发](./howtodev.md)
